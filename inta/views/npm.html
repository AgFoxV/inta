NPM课程概述

npm是javascript的包管理工具,我们可以通过npm查找和复用其他开发者的代码，同样，我们也可以通过npm分享自己的功能模块供其他开发者使用，共同提升开发效率。

第一节：npm简述
npm最初是node package manager的缩写，也就是node.js的包管理工具。现在npm已经发展为javascript的包管理工具。
npm和node绑定在一起安装的，可以通过npm install npm@latest -g更新npm的最新版本
第二节：npm基本使用
本节讲述如何安装包文件，及其相应的版本。如何查找包文件的安装路径。
如何全局安装包文件，及查找相应路径。
安装包文件后，会在当前目录下创建node_modular路径（文件夹）
通过npm ls --depth=0查看包每层的依赖情况
第三节：npm更新包文件
本节主要讲述如何查看已安装的包文件是否为最新版本，及如何更新最新版本的包文件。
第四节：卸载包文件
本节内容很简单，就是使用npm rm commander卸载包文件
第五节：package.json
本节主要讲述package.json的作用，package.json文件的构成，以及在package.json中每一个键的具体意义。如：main代表包的入口文件，因为一个可能会包含多个文件最后将讲述如何快速生成package.json文件。
第六节：发表包文件
本节主要讲述如何发表自己的包文件到npm，供其他开发者使用。
第七节：包和模块
node.js和npm中，包和模块的定义很容易混淆。
包是一个被package.json描述的一个文件夹或路径。
模块可以被node.js中的require引用。
大多数的包都是模块
第八节：npm运行原理
本节主要讲述如何发表自己的包文件到npm，供其他开发者使用。
