EJS模板使用

EJS模板ejs

Sass扩展了CSS3，增加了规则、变量、混入、选择器、继承等特性。并且可生成良好格式化的CSS 代码，易于组织和维护。

模板引擎使用说明
安装模板引擎： npm install ejs
引用模板引擎： var ejs = require('ejs')
express设置模板引擎： app.set('view engine', 'ejs')
设置模板文件： app.engine('html',require('ejs').renderFile)
模板文件引用数据语法：<%%= 数据 %%>
模板文件引用JSON语法：<%%- JSON %%>
模板文件引用代码语法： <%% 代码 %%>
模板文件引用文档语法： <%% include 文档 %%>
